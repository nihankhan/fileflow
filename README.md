# FileFlow

FileFlow is a simple file management application developed in Go. It provides features for uploading files, viewing all files, and downloading files.

## Features

- **Upload a File:** Easily upload files to the server.
- **View All Files:** Browse and view a list of all uploaded files.
- **Download Files:** Download any file from the list.

## Prerequisites

- [Go](https://golang.org/) installed on your machine.

## Getting Started

1. Clone the repository:

   ```
   git clone https://github.com/your-username/fileflow.git
   ```

2. Change into the project directory:

   ```
   cd fileflow
   ```

3. Run the application:

   ```
   go run main.go
   ```

   The application will be accessible at [http://localhost:7000/](http://localhost:7000/).

## Usage

1. **Upload a File:**

   Navigate to [http://localhost:8080/upload](http://localhost:8080/upload) and follow the instructions to upload a file.

2. **View All Files:**

   Navigate to [http://localhost:8080/files](http://localhost:8080/files) to view all uploaded files.

3. **Download Files:**

   Click on a file name on the "View All Files" page to download the file.
